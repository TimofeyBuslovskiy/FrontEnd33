Тег head - це один із основних тегів в мові розмітки HTML, який використовується для включення метаданих і інших важливих інструкцій для браузера, в той час як тег body відповідає за розміщення інформації на сайті.

Термін "лінійний тег" означає, що елементи, які використовують лінійні теги, розташовані один за одним відлівками без обов'язкових переносів або розділювачів.
Приклади лінійних тегів включають в себе такі, як span, a, strong, em, abbr, code
Блоковий тег - це HTML-тег, який визначає блок або контейнер у веб-сторінці, який відображається як самостійний блок і зазвичай розташовується на новому рядку, займаючи всю доступну горизонтальну ширину батьківського елемента.
Приклади блокових тегів включають в себе h1, h6, p, table та інші.
Атрибут - це додаткова інформація, яка додається до HTML або інших мов розмітки для надання додаткової контекстної інформації або параметрів елементам маркування.
Атрибути зазвичай задаються у вигляді пари "ім'я атрибуту" та "значення атрибуту", розділених знаком рівності (=).
